## nvim-config

My Neovim config.

### Setting up

- Install a Neovim build that includes the built-in `vim.pack` plugin manager.
- Start Neovim; missing plugins will be installed by `vim.pack`.
- Update plugins with `:lua vim.pack.update()`.
